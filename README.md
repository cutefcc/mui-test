# mui-pc

- 本地文档调试：yarn start
- 生成文档：yarn docs:build
  - 会生成 dist 目录(放静态服务器)
- 测试：yarn test
- 打包组件：yarn build

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
